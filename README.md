# OMNIKOOK — Dark Kitchen OS

## Visión del Proyecto
OMNIKOOK es un ecosistema técnico de gestión diseñado para optimizar la operación de **Dark Kitchens** y cocinas comerciales de alta eficiencia. Se posiciona como una capa de control radicalmente minimalista y técnica que centraliza la producción, el inventario y la distribución.

## Pilares Core

### 1. Gestión de Producción en Tiempo Real
- **Nodo de Recepción:** Captura de órdenes y flujo de clientes.
- **Nodo de Cocina (Master Monitor):** Distribución de tareas por estaciones de trabajo.
- **Nodo de Cocinero:** Interfaz simplificada para ejecución de tareas.

### 2. Matriz de Insumos y Recetario Técnico
- Descuento automático de stock basado en la composición de cada platillo.
- Alertas de reabastecimiento crítico basada en mínimos configurables.
- Valoración de inventario en tiempo real.

### 3. Capa de Datos y Auditoría
- **Auditoría de Terminal:** Logs detallados de cada movimiento en el sistema para control total de pérdidas y accesos.
- **Dashboard de Control:** Métricas de throughput y rendimiento operativo.

## Roadmap de Integraciones (Omnichannel)
El sistema está diseñado para expandirse mediante conexiones con APIs oficiales para centralizar la entrada de órdenes desde canales externos:
- **WhatsApp Business API:** Recepción de pedidos directos y confirmaciones automáticas.
- **Meta (Messenger / Instagram):** Integración de catálogos y captura de órdenes vía chat.
- **Marketplaces Externos:** Sincronización con plataformas de Delivery de terceros.

## Identidad Visual
- **Estética:** Dark Mode Radical / Brutalismo Minimalista.
- **Paleta:** Base `#0A0A0A`, Acentos `Acid Green (#CCFF00)` y `Electric Blue (#0044FF)`.
- **Tipografía:** `Space Grotesk` (Interfaces) y `JetBrains Mono` (Datos del sistema).
